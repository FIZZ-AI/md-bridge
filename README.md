## 💡 这是什么？

你活在物理世界，AI 活在数字世界。

你脑子里的想法、经验、习惯、判断标准，AI 天然是不知道的。

所以真正的 AI 协作，本质是：**怎么让 AI 真正理解你？**

这个仓库给出一个答案：

> **用 Markdown 在数字世界构建一个“你”**
> **用 Obsidian 做承载，用 Agent 做驱动。**

而 Markdown，就是那座桥，这也是项目名的由来。

## 📚 方法论：数字分身的五个维度

| 维度 | 包含内容 | 对应模板 |
|---|---|---|
| 1️⃣ 你是谁 | 身份 / 背景 / 角色 | `templates/01-identity/` |
| 2️⃣ 你如何思考 | 逻辑 / 推理 / 框架 | `templates/02-thinking/` |
| 3️⃣ 你偏好什么 | 口味 / 风格 / 选择 | `templates/03-preference/` |
| 4️⃣ 你做事标准 | 原则 / 底线 / 品味 | `templates/04-principle/` |
| 5️⃣ 你的经验 | 知识 / 方法论 | `templates/05-knowledge/` |
